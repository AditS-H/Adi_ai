## Q: Tell me about BlitzOS.
**A:** BlitzOS is one of my favorite projects — it's a lightweight operating system kernel I built from scratch in C/C++. I implemented custom memory management with paging and heap allocation, a process scheduler, keyboard driver, and a basic shell. It taught me an incredible amount about how computers actually work at the hardware level.

## Q: Tell me about Cinevox.
**A:** Cinevox is a modern movie discovery platform I built with React, Node.js, and MongoDB. It features real-time search, personalized recommendations based on watch history, and a sleek responsive UI with dark mode. I integrated several third-party APIs for movie data and trailers.

## Q: What is AditAI?
**A:** AditAI is this very project — an AI-powered portfolio assistant that answers questions about me using RAG (Retrieval-Augmented Generation). It has a multi-provider LLM failover system, vector similarity search powered by pgvector, automated GitHub repo indexing, and a clean conversational interface. It's basically me, but available 24/7!

## Q: How does AditAI work?
**A:** AditAI uses a RAG pipeline — when you ask a question, it generates an embedding, searches my vectorized knowledge base for relevant context, and sends that context along with your question to an LLM. The system automatically fails over between five different AI providers (Groq, Gemini, OpenRouter, DeepSeek, Cloudflare) for maximum reliability.
